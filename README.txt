В проекте реализована сортировка статей по периоду публикации, рейтингу.
Возможна настройка пагинации пользователем.
На каждой статье можно оставить свою оценку и комментарий
Статьи можно добавлять в избранное и просматривать понравившееся и избранные статьи
Можно редактировать профиль пользователя: изменять описание, аватар, пароль, почту
Можно создавать/редактировать/удалять статьи авторизованным пользователям

Реализован REST API:

Просмотр без авторизации/с авторизацией

api/v1/articles - выводит все статьи БД, с начальной пагинацией 10 и сортировкой по актуальности
api/v1/category/<slug:cat_slug>/ - выводит все статьи, определённой категории
api/v1/articles/<int:pk/ - Детальный просмотр статьи
api/v1/object/comments - Просмотр комментариев объекта, объектом может быть также экземпляр модели комментария
api/v1/user/<slug:user_slug>/favorite_articles - Просмотр избранных статей определённого пользователя
api/v1/user/<slug:user_slug>/liked_articles/ - Просмотр понравившихся статей определённого пользователя
api/v1/user/<slug:user_slug>/ - Вывод информации об пользователе и написанных им статей

При отправке запроса на изменение пользовательских данных необходимо быть авторизованым через Token.

api/v1/articles/<int:pk>/delete/ - удаление статьи. Права доступа: суперпользователь и автор статьи
api/v1/articles/<int:pk>/update/ - форма редактирования статьи. Права доступа: суперпользователь и автор статьи
api/v1/user/update - В теле запроса необходимо указывать параметры для обновления
api/v1/articles/favorite/ - В теле запроса необходимо отправить pk и название модели статьи (content_type: 'article'), для добавления в избранное
api/v1/vote/ - В теле запроса необходимо отправить pk статьи, название модели статьи(content_type: 'article'), тип голоса like/dislike('vote')
api/v1/object/comment - В теле запроса необходимо указать pk и модель объекта комментирования (content_type: 'article'/'comment'), текст комментария (text)

Авторизация

api/v1/auth/login - авторизация по сессиям
auth/login - авторизация по по JWT токену
